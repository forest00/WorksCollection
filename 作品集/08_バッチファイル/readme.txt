argview.exe
制作時間:約1分

引数の数と引数の中身を表示するプログラムです

以下のソースコードのビルドデータです

[ソースコード]
#include <iostream>

int main(int argc, char **argv)
{
  std::cout << "argc : " << argc <<  std::endl;
  for(int i = 0; i < argc; i++)
  {
    std::cout << i << ", " << argv[i] <<  std::endl;
  }
  return 0;
}
